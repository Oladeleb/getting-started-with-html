What is the Web?

The World Wide Web (WWW) is a collection of HTLM documents. Hyper Text Markup Language (HTML) is the language used to write web pages- glue that ties all file types, images video, pictures, etc. together in a readable format.

HTML

Hyper Text Markup Language (HTML) is use to write webpages. HTML is made up of "text content", what you see, and "mark up" which describes how "what you see" looks like- references to links, images, video, etc. This video gives you a good overview.

Tags and Elements

HTML tags are used to mark the beginning and/or closing of an element. The name of the opening and closing tag must be an exact match. Opening tag determines the HTML output. HTML Elements comprises of opening and closing tags with content.

Inline Vs.. Block Elements

Inline elements end a line and wraps to the next line. While, Block elements renders an invisible box around a text.
